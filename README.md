# Rosswell Company Registration Form

## Project Description

This project is a Company Registration / Service Request Form created using pure HTML. The form collects personal information, company details, service requests, and user agreements.

## Features

* Personal Information Section

  * Full Name
  * Email Address
  * Phone Number

* Company Information Section

  * Company Name
  * Company Website
  * Company Size
  * Industry

* Service Request Section

  * Service Type
  * Preferred Contact Method
  * Appointment Date
  * Appointment Time
  * Budget Range

* Message and Agreement Section

  * Additional Information Text Area
  * Terms and Conditions Agreement

## Technologies Used

* HTML5

## Validation Features

The form uses built-in HTML validation:

* required
* minlength
* maxlength
* pattern
* min
* max

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open the `index.html` file in your web browser.
